# analysis-tomtom-mov-urb
Este dataset tiene información de TomTom Traffic Index y OECD Cities con datos de tráfico, tiempos de desplazamiento, tiempo en detenido en tráfico y las principales ciudades que se analizaron
Se presentan gráficas tipo barra que nos permite visualizar como se relaciona el crecimiento del PIB de una ciudad VS el tiempo que pasan en el tráfico
También tenemos una tabla donde listamos las principales ciudades con su respectivo PIB y los tiempos de traslado y tráfico
